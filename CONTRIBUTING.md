# Contributing to CareerOS

Thank you for your interest in contributing.

CareerOS aims to become an open-source AI Career Intelligence Platform.

## Principles

- Truth before optimization.
- Evidence before generation.
- Explainability over complexity.
- Human-first communication.
- Professional documentation.

## Workflow

1. Fork the repository.
2. Create a feature branch.
3. Make changes.
4. Commit using Conventional Commits.
5. Open a Pull Request.

## Commit Examples

docs: add roadmap

feat: implement interview engine

fix: correct schema validation

refactor: improve reasoning workflow

test: add ATS evaluation tests

## Code Standards

- Follow repository structure.
- Keep documentation modular.
- Explain architectural decisions.
- Write descriptive commit messages.

Thank you for helping build CareerOS.
